Build a full-stack SaaS web app called "RankPilot AI" — an AI-powered content and website optimization tool for SEO, AEO, and GEO.

---

🧠 CORE FUNCTIONALITY:

Users can:

1. Paste content OR a website URL
2. Analyze content for:

   * SEO (Search Engine Optimization)
   * AEO (Answer Engine Optimization)
   * GEO (Generative Engine Optimization)
3. Get:

   * SEO score (0–100)
   * AEO score (0–100)
   * GEO score (0–100)
   * Issues list
   * Suggestions with explanations
4. Click "Fix Everything" to generate optimized content

---

⚙️ TECH STACK:

Frontend:

* React (with Tailwind CSS)

Backend:

* Node.js with Express

Packages to install:

* axios (for fetching URLs)
* cheerio (for parsing HTML)
* cors
* body-parser
* dotenv

AI:

* OpenAI API (use environment variable)

---

📦 INSTALL THESE PACKAGES:

npm install express axios cheerio cors body-parser dotenv

---

🔐 ENV SETUP:

Create .env file:

OPENAI_API_KEY=your_api_key_here

---

🧩 BACKEND IMPLEMENTATION:

Create Express server with 2 endpoints:

---

1️⃣ POST /analyze

Logic:

* Accept input: { content }

* If content starts with "http":
  → Treat as URL
  → Fetch HTML using axios
  → Load HTML with cheerio
  → Remove script/style tags
  → Extract clean text from body

* Else:
  → Use input as content

* Send this prompt to OpenAI:

"Analyze the following content for SEO, AEO, and GEO optimization.

Return ONLY JSON in this format:
{
"seoScore": number (0-100),
"aeoScore": number (0-100),
"geoScore": number (0-100),
"issues": ["..."],
"suggestions": ["... with explanation"]
}

Content:
[CONTENT]"

* Return parsed JSON

* Add error handling:

  * Invalid URL
  * Failed fetch
  * Empty content

---

2️⃣ POST /optimize

Logic:

* Accept input: { content }

* If URL:
  → Fetch + extract content (same as above)

* Send prompt:

"Rewrite and optimize the following content for SEO, AEO, and GEO.

Make it:

* Clear and structured
* Keyword optimized
* Answer-focused (AEO)
* AI-friendly (GEO)
* Include headings, bullet points, FAQs if useful

Content:
[CONTENT]"

* Return optimized content
